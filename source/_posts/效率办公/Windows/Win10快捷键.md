# 应用中的键盘快捷方式

在许多应用中，当你将鼠标指针悬停在某个按钮上时，工具提示中也会显示快捷方式。还有一些应用需要按 Alt 或 F10 键以显示可用的键盘快捷方式。如果菜单中某个字母有下划线，请同时按下 Alt 键和带有下划线的键，而不是选择该菜单项。例如，按 Ctrl + N 可在“画图”中创建新图片。

> 注意
>
> 使用触摸键盘时，您需要按 **Ctrl** 键来查看一些快捷方式。
